2. **Azure for Machine Learning**
   - Use Azure Machine Learning service

## Certification Paths

### AWS Certified Machine Learning - Specialty
- **Cost**: $300
- **Validity**: 3 years
- **Value**: 27% of DS jobs mention AWS
- **Prerequisites**: None formal, but ML and AWS experience recommended

### Microsoft Azure Data Scientist Associate (DP-100)
- **Cost**: $165
- **Validity**: 1 year (annual renewal)
- **Value**: Growing demand, enterprise focus
- **Prerequisites**: Familiarity with Python and ML

### Google Professional ML Engineer
- **Cost**: $200
- **Validity**: 2 years
- **Value**: Highest salaries, cutting-edge tech
- **Prerequisites**: 3+ years experience recommended
