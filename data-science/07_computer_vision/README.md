# Computer Vision

Image processing and computer vision applications.

## Overview

Computer vision combines traditional image processing with deep learning to enable machines to understand visual data. This domain covers foundational image processing techniques and modern deep learning approaches for vision tasks.

## Projects in This Domain

1. **image-processing** (~10 hours)
   - OpenCV fundamentals
   - Image manipulation and transformations
   - Filtering and edge detection
   - Feature extraction
   - Traditional CV techniques

## Learning Path

**Recommended order:**
1. image-processing (traditional CV and OpenCV)
2. **cnn-learning** from 06_deep_learning (deep learning for CV)

**Time investment:** 1-2 months for basics, ongoing for advanced topics

## Prerequisites

- **python-fundamentals** (Python programming)
- **data-science-fundamentals** (NumPy for image arrays)
- **deep-learning-fundamentals** (for CNN-based approaches)

## What You'll Achieve

After completing these projects, you will:
- Process and manipulate images with OpenCV
- Apply filters and transformations
- Detect edges, corners, and features
- Build image classification models
- Understand CNNs for computer vision
- Apply transfer learning for CV tasks
- Deploy CV models

## Key Technologies

**Traditional**: OpenCV, PIL/Pillow, scikit-image
**Deep Learning**: PyTorch Vision, TensorFlow/Keras
**Models**: ResNet, VGG, YOLO, Mask R-CNN
**Tasks**: Classification, detection, segmentation, tracking

## Common Applications

- Image classification
- Object detection
- Semantic segmentation
- Face recognition
- Optical character recognition (OCR)
- Image generation (GANs, diffusion models)
- Video analysis
