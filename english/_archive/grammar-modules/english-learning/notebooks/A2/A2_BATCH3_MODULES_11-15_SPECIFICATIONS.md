## Executive Summary

This document provides complete, detailed specifications for A2 Batch 3: Modules 11-15, focusing on modal verbs. Based on CEFR A2 standards, Cambridge English Profile research, and the existing curriculum progression, these 5 modules systematically introduce and practice modal verbs for advice, obligation, necessity, possibility, permission, and polite requests.

**The 5 Modules:**
1. Module 11: Should/Shouldn't - Giving Advice
2. Module 12: Must/Have To - Obligation and Necessity
3. Module 13: Don't Have To vs Mustn't - Crucial Distinction
4. Module 14: May/Might/Could - Possibility and Permission
5. Module 15: Would Like - Polite Requests and Offers

This specification is ready for direct implementation using the data-driven architecture developed in A2 Batch 2.
