## Executive Summary

This document provides complete, implementation-ready specifications for A2 Batch 5: Modules 21-25, the FINAL batch of A2 level. Based on CEFR A2 standards, Cambridge English Profile research, and analysis of gaps in the existing curriculum (Batches 1-4), these 5 modules systematically complete essential A2 grammar structures and prepare learners for the critical A2→B1 transition.

**The 5 Modules:**
1. Module 21: Zero and First Conditional - If Clauses and Real Possibilities
2. Module 22: Connectors and Linking Words - Because, So, But, Although
3. Module 23: Relative Clauses - Who, Which, That, Where
4. Module 24: Question Formation - All Question Types and Word Order
5. Module 25: A2 Consolidation and Review - Integration and Assessment

This specification completes the full A2 curriculum (25 modules total) and creates a solid foundation for B1 level studies.

## Batch 5 Overview

### Learning Progression

**Building on Previous Batches:**
- **Batch 1 (Modules 1-5):** Past Tense Mastery → Narrative foundation
- **Batch 2 (Modules 6-10):** Future & Time Structures → Planning and sequencing
- **Batch 3 (Modules 11-15):** Modal Verbs → Expressing possibility, obligation, advice
- **Batch 4 (Modules 16-20):** Comparisons & Descriptions → Evaluating and describing
- **Batch 5 (Modules 21-25):** Conditionals, Connections, Questions → **Completing A2 and preparing for B1**

### Why These Topics for A2 Completion?

**Critical Gap Analysis:**

After analyzing CEFR A2 requirements and the existing curriculum, these 5 topics emerged as ESSENTIAL for A2 completion:

1. **Conditionals (Zero + First):** CEFR explicitly requires A2 learners to use zero and first conditionals. Currently missing from curriculum.

2. **Connectors/Linking Words:** A2 learners must connect ideas logically using because, so, but, although. Only "because" briefly covered in Batch 2.

3. **Relative Clauses (Defining):** Basic relative clauses with who/which/that are core A2 requirements for descriptions and definitions.

4. **Question Formation:** While individual question types appear throughout A1/A2, comprehensive question formation mastery is criterial for A2→B1 transition.

5. **Consolidation Module:** Essential for integrating all 25 A2 modules, assessing readiness, and bridging to B1.

### A2 Completion Criteria

After completing all 25 A2 modules, learners will:
- Use all major time frames (past, present, future) reliably
- Express conditions, causes, results, and contrasts
- Describe people, places, and things with relative clauses
- Ask all question types correctly
- Give advice, express obligation, make comparisons
- Communicate effectively in familiar situations
- Be ready to transition to B1 (independent user) level

## MODULE 21: ZERO AND FIRST CONDITIONAL - If Clauses and Real Possibilities

### Module Overview

**Title:** Zero and First Conditional - If Clauses and Real Possibilities
**Subtitle:** If + Present, Present/Will - Real Situations and Possibilities
**Focus Area:** Using conditionals for general truths and real future possibilities
**Estimated Time:** 5-6 hours
**Exercise Count:** 85 exercises

### Learning Objectives

1. Form zero conditional sentences for general truths and facts
2. Form first conditional sentences for real future possibilities
3. Understand when to use zero vs first conditional
4. Use correct verb forms in if-clause and main clause
5. Produce complex sentences with conditional structures

### Why This Topic Is Essential for A2 Completion

**CEFR Requirement:** A2 level explicitly requires "conditionals - zero and 1st" according to Cambridge English Profile and official CEFR descriptors.

**Gap in Current Curriculum:** While basic future forms are covered in Batch 2, conditional structures (if-clauses) are completely absent from Modules 1-20.

**Preparation for B1:** First conditional is foundational for second conditional (B1). Without mastering first conditional, learners cannot progress to hypothetical conditionals.

**Practical Communication:** Conditionals enable expressing cause-effect, making predictions, stating rules, and discussing possibilities - all essential A2 functions.

### Topics Covered

- Zero conditional structure: If + present simple, present simple
- Zero conditional for general truths, scientific facts, and habitual results
- First conditional structure: If + present simple, will + infinitive
- First conditional for real future possibilities and predictions
- Distinguishing when to use zero vs first conditional
- Comma usage with if-clauses (if clause first → comma)
- Alternative word order (main clause first → no comma)
- Common errors: using will in if-clause, confusing zero and first

## MODULE 22: CONNECTORS AND LINKING WORDS - Because, So, But, Although

### Module Overview

**Title:** Connectors and Linking Words
**Subtitle:** Because, So, But, Although - Connecting Ideas Logically
**Focus Area:** Using connectors to express reason, result, contrast, and concession
**Estimated Time:** 5-6 hours
**Exercise Count:** 84 exercises

### Learning Objectives

1. Use "because" and "so" to connect cause and result
2. Use "but" to express contrast between ideas
3. Use "although/even though" for concession (despite contrast)
4. Distinguish between different types of connectors
5. Understand punctuation and word order with connectors

### Why This Topic Is Essential for A2 Completion

**CEFR Requirement:** A2 learners must "link groups of words with simple connectors" (CEFR descriptor). Connecting ideas logically is criterial for A2.

**Gap in Current Curriculum:** Only "because" appears briefly in Module 9 (Batch 2). Comprehensive coverage of logical connectors is missing.

**Preparation for B1:** B1 requires advanced connectors (however, therefore, moreover). Foundation must be built at A2 with basic connectors.

**Discourse Competence:** Without connectors, learners produce disconnected sentences. Connectors enable coherent paragraphs and conversations.

### Topics Covered

- Because - expressing reason/cause (I stayed home because I was sick)
- So - expressing result/consequence (I was sick, so I stayed home)
- Because vs So - choosing based on information order
- But - expressing contrast (I was tired but I went to work)
- Although/Even Though - concession despite contrast
- Punctuation with connectors (comma with so/but, no comma with because)
- Sentence structure and word order with each connector
- Common errors: mixing up because/so, comma splices

## Rationale for Topic Selection and Sequencing

### Why These 5 Topics?

**Research-Based Selection:**

1. **Zero/First Conditionals (Module 21):**
   - CEFR explicitly lists "conditionals - zero and 1st" for A2
   - Cambridge English Profile shows conditionals emerge at A2
   - Completely absent from current curriculum despite being core requirement

2. **Connectors (Module 22):**
   - CEFR descriptor: "can link groups of words with simple connectors"
   - Essential for discourse competence beyond sentence level
   - Currently only "because" covered briefly; comprehensive treatment needed

3. **Relative Clauses (Module 23):**
   - A2 requirement: "relative pronouns for definitions"
   - Critical for descriptive accuracy: "teacher" vs "teacher who taught me"
   - Foundation for B1 distinguishing defining/non-defining clauses

4. **Question Formation (Module 24):**
   - Questions central to CEFR A2: "can ask and answer questions"
   - Scattered across A1/A2 but never systematically integrated
   - Prerequisite for B1 indirect questions

5. **Consolidation (Module 25):**
   - Research: spaced review crucial for retention
   - 25 modules (100+ hours) require systematic integration
   - Assessment function: verify readiness for B1

### Why This Sequence?

**Optimal Learning Progression:**

**Module 21 First (Conditionals):**
- Builds on Batch 2 future forms (going to, will)
- Requires present simple mastery (Batch 1)
- Enables expressing possibilities - motivating for learners
- Foundation for Module 22 (because conditionals show cause-effect)

**Module 22 Second (Connectors):**
- Uses conditionals from Module 21 in examples
- Builds sentence complexity gradually
- Prepares for Module 23 (relative clauses also connect ideas)
- Develops discourse competence before final modules

**Module 23 Third (Relative Clauses):**
- Most complex new structure in Batch 5

- Benefits from connector knowledge (both join clauses)
- Enables sophisticated descriptions in Module 24 questions

**Module 24 Fourth (Questions):**
- Reviews ALL previous A2 grammar in question form
- Questions about conditionals, with connectors, using relatives
- Diagnostic function: reveals gaps in earlier learning
- Prepares for Module 25 assessment

**Module 25 Last (Consolidation):**
- Integrates all 25 modules systematically
- Provides closure to A2 journey
- Assesses readiness for B1
- Motivates continued learning with B1 preview
