# Archive - English Learning Content

This folder contains archived English learning content that has been superseded by the new skill-based organization.

## Contents

### grammar-modules/

**Original location**: `projects/02_intermediate/english-learning/`

**Description**: Comprehensive grammar-focused English learning notebooks organized by CEFR level (A1, A2, B1).

**Content**:
- **A1 Level**: 20 modules, 80 notebooks (4-phase structure per module)
- **A2 Level**: 25 modules, 100 notebooks
- **B1 Level**: 30 modules, 120 notebooks
- **Total**: 75 modules, 400+ notebooks
- **Additional**: Progress tracker, review notebooks

**Pedagogical Approach**:
- 5-phase learning structure (Introduction → Controlled Practice → Meaningful Practice → Communicative Practice → Spaced Review)
- Focus on Form methodology
- Grammar-point focused (each module covers one specific grammar topic)
- Interactive exercises using ipywidgets
- Audio generation for pronunciation
- Spaced repetition system

**Why Archived**:
This content was reorganized into the new skill-based structure (`english/A1/`, `english/A2/`, etc.) that organizes learning by language skills (writing, listening, reading, speaking) rather than by grammar topics. The new approach:
- Aligns better with CEFR skill descriptors
- Integrates AI tools (ChatGPT, Gliglish, Grammarly, ELSA)
- Focuses on real-world communication
- Uses authentic materials (podcasts, articles, videos)
- Emphasizes AI-powered practice and feedback

**Archive Date**: November 16, 2025

**Status**: Complete and functional but superseded by new approach

### Can I Use Both?

Yes! The approaches are complementary:
- **New structure**: For developing speaking, listening, reading, writing skills
- **Archived grammar-modules**: For systematic grammar reference and drill practice

Example workflow:
1. Use new skill-based notebooks for main learning
2. Reference archived grammar modules when you need focused grammar practice
3. Combine AI conversation practice with grammar drills for comprehensive learning
