## Speaking Practice Tools

### 1. Gliglish (⭐ PRIMARY TOOL)

**Why**: Research shows 75%+ improvement in speaking scores over 8 weeks. Real-time grammar corrections, pronunciation feedback, and role-play scenarios.

**Free tier**: 10 minutes daily (perfect for your schedule!)

#### Setup Steps:

1. **Visit**: https://gliglish.com/
2. **Create account**:
   - Sign up with email or Google
   - Select "English" as target language
   - Choose your proficiency level (B1/B2)
3. **Allow microphone access** when prompted
4. **Choose conversation mode**:
   - **Beginner**: Structured topics, slower pace
   - **Intermediate**: Natural conversations, workplace scenarios
   - **Advanced**: Abstract discussions, debates
5. **Daily routine**:
   - Set reminder for same time each day
   - Use all 10 free minutes
   - Focus on workplace scenarios (meetings, presentations, negotiations)
   - Note corrections in your error log

#### Features to Use:

- ✅ Real-time grammar corrections
- ✅ Pronunciation feedback
- ✅ Role-play scenarios (job interviews, client meetings)
- ✅ Conversation history review
- ✅ Personalized improvement suggestions

#### Pro Tips:

- Don't rush through 10 minutes—make them count
- Try different topics daily (Monday: meeting, Tuesday: presentation, etc.)
- Review corrections immediately after session
- Screenshot recurring mistakes for your error log
- When 10 minutes run out, switch to LangAI (below)

## Pronunciation Tools

### 1. ELSA Speak

**Why**: 95%+ accuracy AI speech recognition, distinguishes between American, British, and Australian accents.

**Free tier**: Pronunciation dictionary + basic lessons

#### Setup:

1. Download app: https://elsaspeak.com/ (iOS/Android)
2. Create free account
3. Take placement test
4. Choose accent preference (American/British/Australian)
5. Access free pronunciation dictionary (65,000+ words!)

#### Daily routine:

- Look up 5-10 new words from your reading
- Practice pronunciation
- Get instant feedback on accuracy
- Focus on problematic sounds (th, r, v/w, etc.)

**Pro Tips**:
- Use before AI conversations to warm up
- Practice phrases, not just individual words
- Record yourself, compare to model
- Work on one problem sound per week

## Writing Improvement Tools

### 1. Khan Academy Writing Coach (⭐ PRIMARY)

**Why**: Completely free (Microsoft partnership), detailed feedback on structure, thesis, evidence, style. Coaches you through revision instead of writing for you.

#### Setup:

1. Visit: https://www.khanacademy.org/
2. Create free account
3. Navigate to "Writing" section
4. Access "Writing Coach" tool
5. Start with guided exercises

#### How to Use:

1. **Write first** (no AI help—important!)
2. **Submit** to Writing Coach
3. **Read feedback** on:
   - Thesis clarity
   - Evidence strength
   - Organization
   - Style and tone
4. **Revise** based on suggestions
5. **Resubmit** to see improvement

**Practice schedule**:
- Tuesday/Thursday evenings: 30 minutes
- Sunday mornings: 40 minutes (longer pieces)

**Types to practice**:
- Professional emails
- Project reports
- Proposals
- Opinion essays
- Technical documentation

### 2. Grammarly (Browser Extension)

**Why**: Real-time corrections across all your writing. Learn through immediate feedback.

#### Setup:

1. Visit: https://www.grammarly.com/
2. Create free account
3. Install browser extension (Chrome/Firefox/Edge/Safari)
4. **Important**: Check all installation checkboxes:
   - ✅ Gmail
   - ✅ LinkedIn
   - ✅ Twitter/X
   - ✅ Facebook
   - ✅ Google Docs
   - ✅ All websites

#### How It Works:

- Green underlines: Correctness
- Blue underlines: Clarity
- Purple underlines: Engagement
- Red underlines: Delivery/tone

**Click underlines to**:
- See explanation
- Accept/reject suggestion
- Learn the grammar rule

**Pro Tips**:
- Read explanations, don't just click "Accept All"
- Keep a grammar notebook of recurring corrections
- Set goals in Grammarly (professional, business casual, etc.)
- Review weekly insights for patterns

## Vocabulary Building Tools

### 1. WordUp (⭐ PRIMARY)

**Why**: 2025 Learning Awards Innovation Prize winner. Knowledge Map identifies exactly which of 25,000 ranked words you know. AI conversations with "Lexi," examples from movies/TV/news.

**Free tier**: Extensive free content

#### Setup:

1. Download app: https://wordupapp.co/ (iOS/Android)
2. Create account
3. **Critical**: Complete initial Knowledge Map assessment (15-20 min)
   - This identifies which words you already know
   - Prevents wasting time on known words
   - Prioritizes highest-value unknown words
4. Set daily goal: 20-30 new words/week

#### Daily Routine (10 minutes):

1. **Review yesterday's words** (spaced repetition) - 5 min
2. **Learn 4-6 new words** - 3 min
3. **Practice with "Lexi" AI** - 2 min

#### Features to Use:

- Knowledge Map (shows your vocabulary visually)
- Personalized daily recommendations
- Real examples from movies/TV (context!)
- Spaced repetition system
- AI conversation practice
- Pronunciation audio

**Focus areas**:
- Business vocabulary (meetings, projects, negotiation)
- Phrasal verbs (follow up, bring up, put off)
- Professional collocations (reach agreement, meet deadline)

## AI Tutoring Tools

### 1. ChatGPT

**Why**: Unlimited conversations, explanations, practice exercises. Your 24/7 personal tutor.

**Free tier**: GPT-3.5 completely free

#### Setup:

1. Visit: https://chat.openai.com/
2. Create account (Google sign-in works)
3. Start chatting!

#### Best Uses:

**Vocabulary practice**:
```
I'm studying English and want to expand my vocabulary by using new words in sentences. I will provide one word and a sentence using it. Please check if I've used the word correctly and in a natural context. Identify any errors and offer a corrected version with explanation.

Word: "consolidate"
Sentence: "I need to consolidate my grammar knowledge before the exam."
```

**Grammar explanations**:
```
Act as an English teacher. Explain [Present Perfect] using simple language suitable for a B1 level.

Provide:
- Usage explanation: When and why we use this
- Positive, negative and question forms in a table
- Example sentences
- Common mistakes learners make
- 5 practice sentences for me to complete
```

**Error correction**:
```
Assume the role of an EFL teacher. Based on the following B2-level email, I want you to:

1. Highlight any grammar or spelling mistakes
2. Insert them into a table with three columns: Error | Type of Error | Correction
3. Provide suggestions for overall improvement
4. Point out strong points
5. Suggest more advanced vocabulary

[Paste your writing]
```

**Conversation practice**:
```
Let's role-play in English. I am a project manager presenting a status update and you are my team lead. The scenario is: I need to explain a project delay and propose solutions. Please start the conversation and ask follow-up questions.
```

## Troubleshooting

### "Gliglish won't recognize my voice"

**Solutions**:
- Check microphone permissions in browser
- Use headset with mic (better quality)
- Speak closer to microphone
- Reduce background noise
- Try different browser (Chrome works best)

### "Grammarly not showing up"

**Solutions**:
- Reinstall browser extension
- Check if site is in "Disable list"
- Grant permissions when prompted
- Refresh page after installation

### "WordUp Knowledge Map incomplete"

**Solutions**:
- Set aside 20 focused minutes
- Answer honestly (not too fast)
- Retake if needed—improves recommendations

### "Out of free ChatGPT messages"

**Solutions**:
- Wait for reset (usually 3 hours)
- Switch to Claude temporarily
- Use Google Bard as backup
- Consider paid tier if budget allows ($20/month)

### "EF SET test interrupted"

**Solutions**:
- Tests can be resumed if stopped
- Use computer, not phone (more stable)
- Close other tabs/apps
- Use wired internet if possible
