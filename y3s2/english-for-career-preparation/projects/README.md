# Projects

This folder contains course project materials.

## What to include:
- Project requirements and specifications
- Project planning documents
- Your project work and submissions
- Team collaboration materials (if group project)
- Project presentation slides
- Project reports and documentation

## Organization:
- Create subfolders for each project phase (e.g., `proposal/`, `implementation/`, `final/`)
- Keep project feedback separate
- Maintain version control for important documents
