# English for Career Preparation - Quick Start Guide

Get started with the course in 5 minutes!

## Step 1: Install Requirements (2 minutes)

```bash
# Navigate to course directory
cd y3s2/english-for-career-preparation

# Install required packages
pip install -r requirements.txt
```

## Step 2: Launch Jupyter (1 minute)

```bash
# Start Jupyter Notebook
jupyter notebook

# This will open in your browser at http://localhost:8888
```

## Step 3: Start Learning! (2 minutes)

1. Navigate to the `notebooks/` folder
2. Open `00_setup_introduction.ipynb`
3. Run the first cell (click cell, then press `Shift + Enter`)
4. Follow along!
