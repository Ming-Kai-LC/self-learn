# Assignments

This folder contains all individual assignments and homework.

## What to include:
- Assignment instructions and rubrics
- Your completed assignments
- Feedback and grades received
- Draft versions (optional)

## Organization:
- Create subfolders for each assignment (e.g., `assignment01/`, `assignment02/`)
- Keep original instructions separate from your submissions
- Name files clearly (e.g., `assignment01_submission.docx`, `assignment01_feedback.pdf`)
