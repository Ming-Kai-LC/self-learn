# Lectures

This folder contains all lecture materials from the instructor.

## What to include:
- Lecture slides (.pptx, .pdf)
- Lecture notes from the professor
- In-class handouts
- Recorded lecture links or transcripts

## Organization:
- Name files with lecture number or date (e.g., `week01_topic.pdf`, `lecture_01.pptx`)
- Create subfolders by week/topic if needed
